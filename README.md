# 🎬 Be7iryBest Movies App

A modern web application built with **React** and **Redux** that allows users to **browse, add, and manage movies** with a clean dark-themed UI.

## 🚀 Features
- View a list of movies with details (title, description, poster).
- Add new movies through a simple form.
- Manage global state using **Redux** for predictable updates.
- Dark theme design for a professional look.
- Responsive layout for desktop and mobile.

## 🛠️ Tech Stack
- **React** (UI components)
- **Redux** (state management)
- **React Router** (navigation)
- **CSS/SCSS** (styling)
- **Axios / Fetch API** (optional for backend integration)
